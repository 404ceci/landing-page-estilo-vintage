## ꒰💻꒱ Sobre o projeto

Este projeto consiste no desenvolvimento completo de uma landing page para uma loja de roupas vintage, abrangendo todas as etapas do processo: do design à implementação.

A interface foi inicialmente projetada no Figma, onde foram definidos identidade visual, layout, tipografia e hierarquia de informações. 

Posteriormente, o design foi convertido em código, garantindo fidelidade visual e responsividade.

O foco do projeto foi criar uma experiência visual atrativa e funcional, simulando um cenário real de e-commerce.

## ꒰💻꒱ Processo Criativo

**1. 📝 Conceito**

A landing page foi pensada para transmitir um estilo vintage moderno, combinando elementos clássicos com uma abordagem atual.
A ideia central foi criar uma experiência visual agradável e envolvente, valorizando simplicidade, elegância, nostalgia. 

**2. 🎨 Paleta de Cores**

A escolha das cores foi feita com base no conceito vintage:

- Tons terrosos → transmitem aconchego e nostalgia.
- Bege / Off-white → trazem leveza e equilíbrio visual.
- Marrom escuro → remete ao clássico e à sofisticação.
- Cores suaves → evitam poluição visual e melhoram a experiência do usuário.

**3. 🧠 Decisões de UX/UI**

- Layout limpo e organizado.
- Hierarquia visual clara.
- Uso de espaços em branco para melhorar a leitura.
- Destaque para informações importantes.
- Design focado na conversão.

## ꒰🔄꒱ Alterações e Ajustes

Durante o desenvolvimento, o projeto passou por diversas mudanças e melhorias. Cores, posições, alinhamentos.

Essas alterações foram feitas com foco em:

- Melhorar a experiência do usuário.
- Ajustar detalhes visuais.
- Tornar a interface mais responsiva.
- Corrigir limitações percebidas durante a implementação.

Esse processo faz parte do fluxo real de desenvolvimento, onde o design evolui junto com o código.

## ꒰📌꒱ Objetivos

- Desenvolver uma interface moderna e alinhada ao segmento de moda;
- Criar uma identidade visual coesa do zero;
- Aplicar conceitos de UX/UI no layout;
- Transformar um protótipo em um produto funcional (design → código).
